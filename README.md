## 작문을 도와줘!
----
도움 받기 버튼을 통해 작문된 문장을 리스트로 확인하여 클릭으로 선택하여 TextBox에 추가 할 수 있습니다

## How to run
----
docker-compose up

### Docker build
```
docker build -t $(YOUR_DORCKER_HUB_ID)/react-docker .
```

### Docker run
```
docker run -p 3001:3000 $(YOUR_DORCKER_HUB_ID)/tabtab
```
Now the server is available at http://localhost/3001.
