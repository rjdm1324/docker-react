## 작문을 도와줘!
----
도움 받기 버튼을 통해 작문된 문장을 리스트로 확인하여 클릭으로 선택하여 TextBox에 추가 할 수 있습니다

## How to run
----
* TextBox에 글을 입력 후 도움 받기 버튼을 통해 그 뒤에 이어질 문장을 도움 받을 수 있습니다.
* sample list에서 문장 클릭을 통해 쉽게 textbox로 입력 할 수 있습니다.
* 이어질 문장의 sample 수 와 문장의 단어 수는 counter를 통해 설정 할 수 있습니다.

### Docker build
```
docker build -t $(YOUR_DORCKER_HUB_ID)/react-docker .
```

### Docker run
```
docker run -p 3001:3000 $(YOUR_DORCKER_HUB_ID)/react-docker
```
Now the server is available at http://localhost/3001.
